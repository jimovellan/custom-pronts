# custom-pronts
Almacén de prompts reutilizables organizados por tecnología y uso.

## Estructura de carpetas

```
custom-pronts/
├── dotnet/                          # Prompts para proyectos .NET
│   ├── analyze-solution-structure.md
│   ├── create-api-endpoint.md
│   ├── create-entity-model.md
│   ├── create-repository.md
│   └── create-service.md
└── vue3/                            # Prompts para proyectos Vue 3
    ├── analyze-solution-structure.md
    ├── create-component.md
    ├── create-composable.md
    ├── create-store.md
    └── create-view.md
```

## Uso

Cada archivo `.md` contiene un prompt listo para copiar y pegar en tu asistente de IA preferido. Rellena los campos entre corchetes `[...]` con los detalles específicos de tu proyecto.
