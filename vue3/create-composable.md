# Create Composable (Vue 3)

## Prompt

```
Create a Vue 3 composable for the following use case:

- **Composable Name**: [e.g. useProducts — always prefix with "use"]
- **Responsibility**: [What logic this composable encapsulates]
- **Parameters**: [Input arguments with their types]
- **Returns**: [Reactive references, computed values, or functions exposed by the composable]
- **Dependencies**: [External services, other composables, or stores it relies on]

Requirements:
- Use the Composition API (ref, reactive, computed, watch, lifecycle hooks as needed).
- Use TypeScript with proper type definitions.
- Handle loading and error states when performing async operations.
- Keep side effects (watchers, subscriptions) cleaned up with `onUnmounted`.
- Place the file in the `composables/` folder following the project's naming convention.
```
