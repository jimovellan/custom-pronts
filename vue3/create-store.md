# Create Store (Vue 3 + Pinia)

## Prompt

```
Create a Pinia store for the following use case in a Vue 3 project:

- **Store Name**: [e.g. useProductStore]
- **Responsibility**: [What state and logic this store manages]
- **State Properties**: [List each state property with its type and initial value]
- **Getters**: [Derived/computed values needed from the state]
- **Actions**: [List each action, its parameters, and what it does — including async operations]

Requirements:
- Use the Composition API style (`defineStore` with a setup function).
- Use TypeScript with proper type definitions.
- Handle async actions with loading and error state flags.
- Place the file in the `stores/` folder following the project's naming convention.
- Include comments explaining non-obvious logic.
```
