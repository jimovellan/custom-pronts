# Create Component (Vue 3)

## Prompt

```
Create a Vue 3 component with the following specification:

- **Component Name**: [e.g. ProductCard]
- **Description**: [What this component does and where it is used]
- **Props**: [List each prop with its type, required flag, and default value]
- **Emits**: [List each event the component can emit with its payload type]
- **Slots**: [List any slots if applicable]
- **Internal State**: [Any reactive state managed inside the component]

Requirements:
- Use the Composition API with `<script setup>` syntax.
- Define props with `defineProps` and emits with `defineEmits`.
- Use TypeScript with proper type definitions.
- Keep the component focused on a single responsibility.
- Add basic styling using scoped CSS or the project's existing styling approach (Tailwind, etc.).
- Include a brief JSDoc description at the top of the script block.
```
