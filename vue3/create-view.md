# Create View (Vue 3)

## Prompt

```
Create a Vue 3 view (page component) with the following specification:

- **View Name**: [e.g. ProductListView]
- **Route**: [e.g. /products]
- **Description**: [What this view displays and its main purpose]
- **Data Needed**: [What data must be fetched or received via route params/query]
- **Components Used**: [Child components this view composes]
- **User Interactions**: [Actions the user can perform on this view]

Requirements:
- Use the Composition API with `<script setup>` syntax and TypeScript.
- Fetch data using a composable or Pinia store action — avoid direct API calls inside the view.
- Handle loading and error states in the template.
- Register the route in the router file (`src/router/index.ts`).
- Apply responsive layout consistent with the rest of the application.
```
